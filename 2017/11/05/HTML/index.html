HTML

{% note info %}HTML 是接触前端的第一课 {% endnote %}

  • WHAT

    • HTML（HyperText Markup Language） 超文本标记语言
    • 万维网（WWW）中，用URL来做资源的定位，用HTTP/HTTPS作为访问的协议，用HTML来做访问时的导航（通过img指向图片 video指向视频等）
    • HTML构成网页的骨架结构。表明不同区域在页面中的含义及作用（语义化）

  • 组成

    • 元素(Element)
      • <p>这是一个P标签</p>
      • <p></p> 是一对闭合标签
      • 这是一个P标签 是<p></p>的内容（content）
      • 标签 + 内容 构成了元素
    • 标签
      • 分类
        • 闭合标签
        • 空标签
      • 属性
        • 标签拥有自己的属性，表示特定的功能
    • 元素
      • 分类
        • 块元素(Block)
          • <div></div>、<p></p>、<h1>-<h6>、<ol><ul><li>、<dl><dt><dd>、<table><thead><tbody><tfoot><caption><tr>、<form>
        • 内联元素(inline)
          • <a>、<img /><input />、<span><i>

  • 实例

    • <!DOCTYPE html> <!-- 文档类型 -->
    • <html></html> <!--根元素-->
      • 包含 head、body标签
    • <head></head> <!-- 头文件 -->
      • 包含title、meta等描述网页信息的标签和link、style、script等引入资源的标签
    • <body></body> <!-- 网页正文 -->
      • 是网页的正文部分

  • 语义化

    • 语义化标签
      • <header></header>
      • <nav></nav>
      • <section></section>
      • <article></article>
      • <aside></aside>
      • <footer></footer>
      • <address></address>
    • 作用
      • 使页面结构明确，利于浏览器的解析、搜索引擎的搜索和页面的SEO
      • 易于阅读，更快的获取所需信息。
      • 优化用户体验。(盲人设备、CSS加载失败)
